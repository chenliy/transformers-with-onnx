- cudatoolkit               10.1
- cudnn                     7.6.5
- onnx                      1.7.0
- onnxconverter-common      1.7.0
- onnxruntime-gpu           1.3.0
- onnxruntime-tools         1.3.0
- keras2onnx           1.7.0
- transformers              up-to-date
